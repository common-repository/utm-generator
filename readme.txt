== UTM Code Generator Wordpress Plugin Features ==

* The plugin requires only your Google Analytics Tracking ID if you want to install tracking code (if tracking code not exists).
* Easy form to build a free UTM link from your existing Wordpress or other website link.
* Good help included to explain the building form.
* The plugin adds menu in the admin bar for fast building.
* The plugin helps you to store default UTM (post meta) values for every post or page in the add/edit page for posts or pages.
* <strong>Full control</strong>: enable/disable admin bar option.
* <strong>Full control</strong>: enable/disable plugin for posts, pages or both.
* <strong>Full control</strong>: enable/disable installing Google Analytics Tracking code.
* <strong>Full control</strong>: the position of the Google Analytics Tracking code (header or footer).
* <strong>Full control</strong>: set default UTM values for builder wizard.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `UTM Generator` from Plugins page.

The plugin will add a new menu called `UTM Generator`. Goforward using the plugin from here!
